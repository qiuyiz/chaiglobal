# CHAI Global — Christians in AI

Open-source tools for redemptive technology development. A partnership between [CHAI Global](https://chaiglobal.ai) and [FaithTech](https://faithtech.com).

## Projects

### [Selah for Claude Code](selah-claude-code/)

Drop two files into any project to give Claude Code a redemptive operating posture. Includes a `/selah` breath prayer command that pauses your session with context-aware Scripture.

**Quick start:** Copy `CLAUDE.md` and `.claude/commands/selah.md` into your project. Type `/selah`.

### [Selah for OpenClaw](selah-openclaw/)

A contemplative AI agent workspace for OpenClaw. Transforms your agent into an unhurried co-creator with automatic Selah rhythm, 3RC discernment, sabbath mode, and morning intentions.

**Quick start:** Copy the folder to `~/.openclaw/agents/selah/`. Edit `USER.md` with your preferences.

## The Framework

Both projects implement FaithTech's [Redemptive Technology Playbook](https://faithtech.com/playbook):

- **Three Ways of Technology**: Reckless → Responsible → Redemptive
- **3RC Discernment**: Reject, Receive, Reimagine, Create
- **4D Cycle**: Discover, Discern, Develop, Demonstrate
- **Kenotic Design**: Self-emptying as a design principle — do less, serve more
